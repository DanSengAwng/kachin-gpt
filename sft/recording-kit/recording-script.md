# Recording Script (template)

**To the session organizer:** this template deliberately does NOT contain
machine-written Kachin. Fill the blank slots with sentences chosen by
native speakers -- everyday speech, plus passages in the reading style
your community actually uses. Keep sentences 5-20 words.

**To the speaker:** read naturally, at your normal pace, as if reading to
a friend. Read each item twice.

## Block A -- Termbank items (verified entries only)

Read each entry from `termbank/termbank.csv` that has `verified: true`,
in order, twice each. (If nothing is verified yet, verify first -- this
kit refuses to put unreviewed machine text in speakers' mouths.)
